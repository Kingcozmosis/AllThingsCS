Privacy Policy

Last Updated: March 22, 2026

```

1. Information Collected

The Bot may collect and temporarily process the following information provided by Discord:

  • Server (Guild) ID
  • Channel IDs used for setup commands
  • Role IDs used for notifications
  • User IDs for command interaction

This information is only used to operate the Bot’s features, such as posting news, delivering CS2 updates, managing server configurations, and processing commands. No personal information outside of what Discord provides is collected.

2. Data Storage

The Bot stores minimal configuration data such as:

  • Server configuration settings
  • Selected channels
  • Selected roles
  • Last posted news/update links

This data is stored locally in configuration files strictly for the purpose of running the Bot’s features.

3. Data Usage

Collected data is used solely for:

  • Providing bot functionality
  • Delivering configured updates
  • Responding to commands
  • Maintaining server settings

Data is never sold, shared, or used for advertising purposes.

4. Third-Party Services

The Bot may retrieve information from third-party sources including:

  • BO3.gg
  • Dust2.us
  • Steam Community
  • CS2 API services

These services may have their own privacy policies and terms.

5. Data Removal

Server administrators can remove stored data at any time by:

  • Disabling bot features using setup commands
  • Removing the bot from the server

Once removed, configuration data related to that server may be deleted.

6. Security

Reasonable efforts are made to protect stored configuration data. However, no system is completely secure.

7. Changes to this Policy

This Privacy Policy may be updated periodically. Continued use of the Bot constitutes acceptance of any changes.

8. Contact

For questions regarding this Privacy Policy, please contact the developer through the GitHub repository .
